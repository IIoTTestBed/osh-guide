# 5. Kompatibilität und Modularisierung

Die Wahl der Lizenz, wie auch die Lizenzen verwendeter Komponenten, wirken sich maßgeblich auf die **Kompatibilität** und **Modularisierung** aus.

!!! info "Weiterführende Informationen"
    Siehe [Kapitel 4 – Lizenzierung](04-lizenzierung.md) für eine detaillierte Übersicht der Lizenztypen und deren Copyleft-Wirkung.

Strikte Lizenzen (reziproke Lizenzen) verlangen, dass abgeleitete Werke unter derselben oder einer kompatiblen Lizenz veröffentlicht werden. Dies kann die Integration mit anderen Komponenten erschweren, stellt aber sicher, dass das Werk dauerhaft offen bleibt.

Freizügige Lizenzen erleichtern die Kombination mit anderen Komponenten, ermöglichen aber auch eine proprietäre Weiternutzung.

Als Kompromiss wird daher oft eine **schwache Lizenzierung** gewählt.
